# SERP·X

SERP analyzer for Greece — DR, Ref Domains, and mobile Core Web Vitals (LCP, INP, CLS) for top 10 results.

## Deploy to Vercel

1. Go to [vercel.com](https://vercel.com) and sign in with GitHub
2. Click **"Add New Project"** → Import `serpx`
3. In **Environment Variables** add:
   - `AHREFS_API_KEY` = your Ahrefs API key
   - `CRUX_API_KEY` = your CrUX API key
4. Click **Deploy**

That's it.
